========================================
Birthday Video Automation - Simple Guide
========================================

STEP 1: Start Chrome
--------------------
Double-click: step1-start-chrome.bat

- Chrome will open
- Login to Twitter
- Keep Chrome open

STEP 2: Run Automation
--------------------
Double-click: step2-auto-run.bat

1. Start screen recording (Win+G)
2. Adjust volume
3. Press Enter
4. Wait ~90 seconds
5. Stop recording

Done!

What happens (90 seconds):
--------------------
1. Opening (4s)
2. Twitter search (6s)
3. Maid costume tweet (8s) + like
4. JR Otsuka station ad (8s) + highlight + like
5. TikTok video (10s) + play with sound
6. idolfunch tweet (10s) + video play + like
7. Finale (6s)

Files:
--------------------
step1-start-chrome.bat              <- Run this first
step2-auto-run.bat                  <- Run this second
birthday-automation-logged-in.js    <- Automation script
README-EN.txt                       <- This file

Troubleshooting:
--------------------
- Chrome not starting? Close all Chrome windows first
- Login screen appears? Complete Step 1 login
- No video? Check volume/sound
- Error? Check Chrome is on port 9222
