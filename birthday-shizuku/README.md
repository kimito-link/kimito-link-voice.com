# 🎉 蒼凪しずく 生誕祭動画プロジェクト

## 📋 概要

Chrome DevTools MCPを使って、Twitterや TikTokを自動操作しながら録画して、蒼凪しずくさんの生誕祭動画を作成するプロジェクトです。

## 🎬 シナリオ内容

### 1. Twitter検索シーン (0:00-0:10)
- Twitterを開く
- 検索ボックスに「蒼凪しずく 生誕祭 楽しみ！🎉」と1文字ずつゆっくり入力
- エンターキーで検索実行

### 2. メイド服ツイート (0:10-0:20)
- https://x.com/flap_shizuku/status/1988950811075125652
- しずくちゃんのメイド服写真ツイートを表示
- ゆっくりスクロールして画像を見る
- ❤️ いいねボタンをクリック

### 3. JR大塚駅広告 (0:20-0:30)
- https://x.com/flap_up_idol/status/1988510278448017445
- JR大塚駅の生誕祭広告を表示
- 広告をゆっくり眺める
- ❤️ いいねボタンをクリック

### 4. TikTok動画 (0:30-0:40)
- https://www.tiktok.com/@idolfunch/video/7509897290023177489
- TikTok動画を再生
- 5秒間視聴
- ❤️ いいねボタンをクリック

### 5. idolfunchツイート動画 (0:40-0:50)
- https://x.com/idolfunch/status/1942732395515633764
- Twitter動画を再生
- 動画を視聴

### 6. 最終メッセージ (0:50-1:00)
- 特製アニメーションページを表示
- 「蒼凪しずく 生誕祭 楽しみ！✨ おめでとう ✨」

## 🛠️ 使用方法（2025-11-13更新）

### 🎬 Puppeteer自動化（推奨）

#### ステップ1: Chromeをデバッグモードで起動
```bash
.\step1-start-chrome.bat
```
- Chromeが起動します
- **Twitterにログイン**してください
- Chromeは開いたままにします

#### ステップ2: 自動実行
```bash
.\step2-auto-run.bat
```
1. Win+G で画面録画を開始
2. 音量を調整
3. Enterを押す
4. 約90秒待つ
5. 録画停止

### 📂 ファイル一覧

- `step1-start-chrome.bat` - Chromeをデバッグモードで起動
- `step2-auto-run.bat` - 自動化スクリプトを実行
- `birthday-automation-logged-in.js` - メインの自動化スクリプト（Puppeteer）
- `birthday-slideshow.html` - スライドショー版（埋め込みでログイン不要）

### ⚠️ 既知の問題（2025-11-13時点）

1. **Twitterログイン画面が出る**
   - 原因: ログイン状態が保存されていない
   - 対策: step1でログインしてからstep2を実行

2. **TikTok/Twitter動画の音が出ない**
   - 原因: ブラウザの自動再生ポリシー
   - 対策: Puppeteerで動画を自動クリックして再生

3. **文字エンコーディング問題**
   - 原因: batファイルの日本語が文字化け
   - 対策: 英語名ファイル（step1, step2）を使用
