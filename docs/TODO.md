## 📝 技術的な決定事項

### 採用技術
- **フロントエンド**: HTML + CSS + JavaScript（Vanilla）
- **バックエンド**: Node.js + Express
- **データベース**: Supabase（PostgreSQL）
- **認証**: Twitter OAuth 2.0
- **ストレージ**: ローカル（将来的にCloudinary検討）
- **デザイン**: ゆっくりスタイル + パステルカラー

### 不採用技術（理由）
- ❌ **11ty**: 静的サイトジェネレーターは不要（動的機能が必要なため）
- ❌ **React/Vue**: オーバースペック（Vanillaで十分）
- ❌ **Stripe Connect**: 実装コストが高い（決済リンクで代替）
