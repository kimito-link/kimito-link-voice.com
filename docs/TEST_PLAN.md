## ✅ 機能テスト

### 1. 認証・ログイン機能

#### 1.1 Twitter OAuth ログイン
- [ ] ログインボタンをクリックしてログインモーダルが表示される
- [ ] フォロー必須アカウントが正しく表示される（@streamerfunch, @idolfunch）
- [ ] 「ログインして始める」ボタンでTwitter認証画面に遷移する
- [ ] Twitter認証後、正しくリダイレクトされる
- [ ] ログイン後、ダッシュボードが表示される
- [ ] ユーザー情報（アバター、名前）が正しく表示される

#### 1.2 ログアウト機能
- [ ] ログアウトボタンをクリックしてログアウトできる
- [ ] ログアウト後、TOPページに戻る
- [ ] セッション情報が正しくクリアされる

#### 1.3 マルチアカウント対応
- [ ] アカウント履歴がlocalStorageに保存される
- [ ] ドロップダウンメニューで履歴が表示される
- [ ] 「違うアカウントでログインする」ボタンが動作する

### 2. ダッシュボード機能

#### 2.1 依頼者モード
- [ ] 「依頼する」タブが表示される
- [ ] 「音声アップロード」タブが表示される
- [ ] 「感謝」タブが表示される
- [ ] ウェルカムメッセージにユーザー名が表示される
- [ ] ロールスイッチで声優モードに切り替えられる

#### 2.2 声優モード
- [ ] 「完了した案件」カードが表示される
- [ ] 「今月の収益」カードが表示される
- [ ] 「音声アップロード」タブが表示される
- [ ] 「感謝」タブが表示される
- [ ] ロールスイッチで依頼者モードに切り替えられる

### 3. 音声アップロード機能

#### 3.1 ファイル選択
- [ ] ファイル選択ボタンでファイル選択ダイアログが開く
- [ ] 対応形式（MP3, WAV, OGG, M4A）のみ選択できる
- [ ] ファイルサイズが25MB以下であることを確認
- [ ] ファイル選択後、プレビューが表示される

#### 3.2 アップロード
- [ ] タイトル入力欄に入力できる
- [ ] 説明文入力欄に入力できる
- [ ] カテゴリを選択できる
- [ ] 「アップロード」ボタンでアップロードが開始される
- [ ] プログレスバーが表示される
- [ ] ゆっくりローディングアニメーションが表示される
- [ ] アップロード完了後、成功メッセージが表示される

### 4. 感謝タブ機能

#### 4.1 感謝メッセージ投稿
- [ ] 感謝の文章を入力できる
- [ ] プレビューにハッシュタグとメンションが表示される
- [ ] 「拡散ボタン」でTwitter投稿画面が開く

#### 4.2 感謝メッセージ一覧
- [ ] カード形式で一覧表示される
- [ ] アバター、名前、フォロワー数が表示される
- [ ] 投稿日時が表示される
- [ ] 自分の投稿には編集・削除ボタンが表示される

### 5. TOPページ機能

#### 5.1 ヒーローセクション
- [ ] キャッチコピーが表示される
- [ ] CTAハイライト「今なら初回依頼500円から！」が脈動する
- [ ] 「推しに想いを届ける」ボタンでログインモーダルが開く
- [ ] 「声優として活動する」ボタンでログインモーダルが開く

#### 5.2 声優カード
- [ ] 声優カード1（@streamerfunch）の情報が表示される
- [ ] 声優カード2（@idolfunch）の情報が表示される
- [ ] アバター画像が表示される
- [ ] フォロワー数が表示される
- [ ] プロフィール文が2行まで表示される（それ以上は省略）
- [ ] タグが表示される
- [ ] 料金が表示される

#### 5.3 FAQセクション
- [ ] 8つの質問が表示される
- [ ] 質問をクリックするとアコーディオンが開く
- [ ] 他の開いている質問は自動的に閉じる
- [ ] アイコンが+から×に変わる
- [ ] 回答が表示される

### 6. エラーページ

#### 6.1 404ページ
- [ ] 存在しないURLにアクセスすると404ページが表示される
- [ ] ゆっくりキャラクターが表示される
- [ ] 「ホームに戻る」ボタンでTOPページに戻る
- [ ] 「前のページに戻る」ボタンでブラウザバックする

## ⚡ パフォーマンステスト

### ページ読み込み速度
- [ ] TOPページが3秒以内に読み込まれる
- [ ] ダッシュボードが3秒以内に読み込まれる
- [ ] 画像が遅延読み込み（lazy loading）される
- [ ] WebP形式の画像が使用されている

### API レスポンス
- [ ] プロフィール取得APIが1秒以内に応答する
- [ ] キャッシュが正しく機能している
- [ ] Twitter APIのレート制限対策が機能している

### JavaScript パフォーマンス
- [ ] ページ遷移が瞬時
- [ ] アニメーションがスムーズ（60fps）
- [ ] メモリリークがない
