# 🎬 KimiLink Voice - 動画撮影スクリプト

作成日: 2025/11/22

---

## 🎥 動画構成（60-90秒）

### シーン1: オープニング（10秒）
**画面**: ロゴアニメーション or 君斗りんくのイントロ
**音声**:
```
こんにちは、君斗りんくです。
今日は、新しいプロジェクトを発表します。
```

---

### シーン2: プロジェクト紹介（15秒）
**画面**: KimiLink Voiceのトップページをスクロール
**音声**:
```
その名も「KimiLink Voice」

声で想いを届ける、
新しいマッチングプラットフォームです。

配信者に、アイドルに、クリエイターに。
オリジナルボイスで応援の気持ちを形にできます。
```

---

### シーン3: きっかけ（15秒）
**画面**: ライブ映像 or 思い出の写真（@flap_shizuku）
**音声**:
```
このプロジェクトのきっかけは、
しずくちゃんの一言でした。

「声を使う者同士、頑張ろうね」

ステージで魂を燃やした後、
少し枯れた声で言ってくれたこの言葉。

声には、文字だけでは伝わらない温度がある。
そう気づかせてくれました。
```

---

### シーン4: 機能紹介（20秒）
**画面**: デモ画面（実際のサイトを操作）
**音声**:
```
使い方はとってもシンプル。

【依頼する人は】
1. 声優さんを選んで
2. 台本を書いて
3. 依頼するだけ

【声優として活躍したい人は】
1. Twitterで簡単登録
2. 料金を自分で設定
3. すぐに活動開始

プロもアマチュアも、みんなが輝ける場所です。
```

---

### シーン5: 技術的特徴（10秒）
**画面**: コード or ダッシュボード画面
**音声**:
```
セキュリティも万全。
Twitter OAuth 2.0で安全ログイン。

スマホでもPCでも、
どこからでも使えます。
```

---

### シーン6: コラボメンバー紹介（10秒）
**画面**: コラボメンバーカードを表示
**音声**:
```
現在、素敵なクリエイターさんとコラボ中！

一緒に盛り上げてくれる仲間も募集しています。
```

---

### シーン7: クロージング（10秒）
**画面**: ロゴ + Coming Soon
**音声**:
```
KimiLink Voice
近日、ベータ版公開予定。

君と声でつながる、新しい世界へ。

最新情報は @streamerfunch をフォロー！
```

---

## 📸 撮影のポイント

### 必要な素材
1. ✅ **トップページのスクリーンショット/動画**
   - ゆっくりスクロールして全体を見せる
   - 声優カードの表示
   - レスポンシブデザインの確認

2. ✅ **ダッシュボード画面**
   - ログイン後の画面
   - タブの切り替え
   - スムーズな動作

3. ✅ **コラボメンバーカード**
   - しずくちゃん
   - コタのAI紀行さん
   - フォロワー数が正しく表示されている

4. ✅ **モバイル画面**
   - スマホでの表示
   - タブレットでの表示
   - レスポンシブ対応の確認

5. 📸 **思い出の写真/動画**
   - しずくちゃんとのライブ
   - コラボ時の写真
   - 開発中の様子

## 🎤 音声収録のコツ

### トーン
- **最初**: ワクワク感、明るく
- **きっかけ**: 少ししみじみ、温かく
- **機能紹介**: 元気に、テンポよく
- **クロージング**: 希望に満ちた感じ

### ポイント
1. **間を大切に**
   - 急がず、ゆっくり話す
   - 大事な言葉の前後に間を入れる

2. **感情を込める**
   - 特に「声を使う者同士、頑張ろうね」の部分
   - 思い出を語るように

3. **笑顔で話す**
   - 声に表情が乗る
   - ポジティブな印象を与える

## 🎬 撮影チェックリスト

### 事前準備
- [ ] サーバー起動（`npm start`）
- [ ] ブラウザでサイト確認（http://localhost:3000）
- [ ] 画面録画ソフト起動
- [ ] マイク音量確認
- [ ] 背景音楽準備（必要に応じて）

### 撮影時
- [ ] 画面がクリアに映っているか
- [ ] 音声がクリアに録音できているか
- [ ] スクロール速度は適切か
- [ ] 文字が読みやすいか

### 撮影後
- [ ] 音声の編集（ノイズ除去）
- [ ] BGM追加
- [ ] 字幕追加（必要に応じて）
- [ ] エフェクト追加（必要に応じて）

## 🎨 編集のポイント

### BGM
- **オープニング**: 軽快で明るい曲
- **きっかけ**: 温かく、感動的な曲
- **機能紹介**: テンポの良い曲
- **クロージング**: 希望に満ちた曲

### トランジション
- シーンの切り替えは自然に
- フェードイン/アウトを効果的に使う

### テキスト/字幕
- 重要なキーワードを強調
- 読みやすいフォントサイズ
- 背景とのコントラスト確保
