## はじめに

KimiLink Voiceへようこそ! 🌟

このプラットフォームは、クリエイターとファンをつなぐ音声配信サービスです。
Twitter連携により、簡単にレビューを共有し、コミュニティを広げることができます。

### 必要なもの
- Twitterアカウント
- モダンなWebブラウザ（Chrome、Firefox、Safari、Edge推奨）
- インターネット接続

## ログイン方法

### ステップ1: Twitterでログイン

1. KimiLink Voiceのトップページにアクセス
2. 「Twitterでログイン」ボタンをクリック
3. Twitter認証画面が表示されます
4. 「アプリを認証」をクリック

#### 2. 君斗りんく@アイドル応援
- Twitter: [@idolfunch](https://twitter.com/idolfunch)
- アイドルを応援するアカウント

両方のアカウントをフォローすると、メインプラットフォームが表示されます！ 🎉

## ダッシュボードの使い方

### 画面構成

ダッシュボードは以下のセクションで構成されています：

#### 1. プロフィールカード
- あなたのTwitterアイコン
- アカウント名
- フォロワー数・フォロー中の数

#### 2. 実績カード
- **音声配信数**: あなたが投稿した音声の数
- **レビュー数**: あなたが獲得したレビューの数
- **リーチ数**: あなたのコンテンツが届いた人数

#### 3. フォロー状態カード
- 必須アカウントのフォロー状態を確認できます
- ✓ マークがついていればOK!

#### 4. コラボオプションカード
- コタのAI紀行さんとのコラボ機能
- 詳細は[コラボ機能の使い方](#コラボ機能の使い方)へ

### ナビゲーション

画面上部のメニューから各セクションに移動できます：

- **ダッシュボード**: ホーム画面
- **レビュー**: レビュー・感想の閲覧と投稿
- **実績**: 詳細な統計情報
- **オプション**: 各種設定

## レビューの投稿方法

### Twitter連携レビューシステム

KimiLink Voiceでは、Twitterを使ってレビューを投稿・共有します。

### レビューを投稿する

#### 方法1: プラットフォームから投稿

1. ダッシュボードの「レビュー」セクションへ移動
2. 「レビューをツイート」ボタンをクリック
3. Twitter投稿画面が開きます
4. 感想を入力（ハッシュタグは自動で追加されます）
5. 「ツイート」ボタンをクリック

#### 方法2: Twitterから直接投稿

1. Twitterを開く
2. 新規ツイート作成
3. 感想を入力
4. **必須**: `#kimitoLinkVoice` ハッシュタグを追加
5. メンション: `@streamerfunch` または `@idolfunch` を追加（推奨）
6. ツイート

### レビュー投稿のコツ

#### 良いレビューの例:
```
KimiLink Voiceで音声配信してみました！
操作が簡単で、声がしっかり届く感じがいい！
#kimitoLinkVoice @streamerfunch
```

#### ハッシュタグの使い方:
- **必須**: `#kimitoLinkVoice`
- **推奨**: アカウントのメンション
- **オプション**: その他関連タグ

### レビューの確認

- プラットフォームの「レビュー」セクションで、`#kimitoLinkVoice` タグがついた投稿が表示されます
- 他のユーザーのレビューも確認できます
- いいね・リツイートで交流しましょう！

## よくある質問

### アカウント・ログイン

**Q: フォローを解除したらどうなる？**
A: 必須アカウントのフォローを解除すると、プラットフォームにアクセスできなくなります。再度フォローすればアクセス可能です。

**Q: ログアウトするには？**
A: 画面右上のユーザー名の横にある「ログアウト」ボタンをクリックしてください。

**Q: 複数のTwitterアカウントで利用できる？**
A: 可能です。それぞれのアカウントで別々にログインしてください。

### レビュー・投稿

**Q: レビューが表示されない**
A: 以下を確認してください：
- `#kimitoLinkVoice` ハッシュタグが正しく入力されているか
- ツイートが公開設定になっているか
- 反映まで数分かかる場合があります

**Q: レビューを削除したい**
A: Twitterでツイートを削除してください。プラットフォームからも数分後に削除されます。

**Q: 過去のレビューを見たい**
A: 「レビュー」セクションのタイムラインをスクロールするか、Twitterで `#kimitoLinkVoice` を検索してください。

### 実績・統計

**Q: 実績はどのように計算される？**
A: 
- **音声配信数**: アップロードした音声の数
- **レビュー数**: `#kimitoLinkVoice` タグがついたツイート数
- **リーチ数**: ツイートのインプレッション数の合計

**Q: 統計が更新されない**
A: ページを再読み込みしてください。それでも更新されない場合は、数時間後に再度確認してください。

### トラブルシューティング

**Q: ログインできない**
A: 以下を試してください：
1. ブラウザのキャッシュとCookieをクリア
2. 別のブラウザで試す
3. Twitterログイン状態を確認
4. それでも解決しない場合は [@streamerfunch](https://twitter.com/streamerfunch) へDM

**Q: 画面が正しく表示されない**
A: 
- ブラウザを最新版に更新してください
- 推奨ブラウザ: Chrome、Firefox、Safari、Edge

**Q: スマホで使えない**
A: KimiLink Voiceは完全レスポンシブ対応です。スマホでも快適に使用できます。

### その他

**Q: プライバシーは守られる？**
A: はい。プラットフォームではTwitterの公開情報のみを使用します。非公開情報は取得しません。

**Q: 有料機能はある？**
A: 基本機能は無料です。コラボ機能のみ有料（¥30,000）です。

**Q: 機能のリクエストは？**
A: ぜひ！[@streamerfunch](https://twitter.com/streamerfunch) または [@idolfunch](https://twitter.com/idolfunch) へご連絡ください。
