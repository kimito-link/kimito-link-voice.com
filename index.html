Chrome プロフィール機能

1
Chromeブラウザでプロフィールを追加

Chromeブラウザ右上のプロフィールアイコン（丸いアイコン）をクリック → 「追加」または「別のユーザーを追加」を選択

2
新しいChromeウィンドウが開きます

完全に独立したブラウザ環境として動作します

3
それぞれのプロフィールでログイン

各ウィンドウで異なるアカウントにログインできます

ヒント

  • 重要： KimiLink Voiceのプロフィールメニューではなく、Chromeブラウザ自体のプロフィール機能です（アドレスバーの右側、ブラウザの右上隅にあります）
  • プロフィールごとに名前とアイコンをカスタマイズできます
  • ブックマーク、履歴、パスワードも完全に分離されます
  • 複数のウィンドウを同時に開いて作業できます

Firefox コンテナタブ機能

1
Firefoxの拡張機能をインストール

Firefoxブラウザで「Multi-Account Containers」をFirefoxアドオンストアから無料でインストール

2
新しいコンテナタブを作成

タブの「+」ボタンを長押し → コンテナを選択

3
各コンテナで異なるアカウントにログイン

コンテナごとに独立したCookieを持ちます

ヒント

  • コンテナごとに色分けして管理できます
  • 同じウィンドウ内で複数アカウントを使い分け可能
  • 「仕事用」「プライベート用」など目的別に分けられます

Edge プロフィール機能

1
Edgeブラウザでプロフィールを追加

Edgeブラウザ右上のプロフィールアイコン → 「プロフィールの追加」を選択

2
新しいEdgeウィンドウが開きます

Chromeと同様に独立した環境で動作します

3
Microsoft アカウントと連携可能

設定を複数デバイス間で同期できます

ヒント

  • 重要： KimiLink Voiceのプロフィールメニューではなく、Edgeブラウザ自体のプロフィール機能です（アドレスバーの右側、ブラウザの右上隅にあります）
  • 職場アカウントと個人アカウントを分けて管理
  • EdgeとChromeは同じChromiumベース
  • Windows 11との統合が優れています

Safari プライベートウィンドウ

1
Safariブラウザでプライベートウィンドウを開く

Safariブラウザのメニュー「ファイル」 → 「新規プライベートウィンドウ」（⌘+Shift+N）

2
通常ウィンドウとプライベートウィンドウを両方使用

それぞれで異なるアカウントにログインできます

3
注意：プライベートウィンドウを閉じるとログアウト

セッションは保持されません

ヒント

  • MacユーザーにはChrome/Firefoxの方が便利
  • プライベートウィンドウは一時的な使用に適しています
  • iOS/iPadOSでも同様の方法が使えます
