基本的にMP3形式で納品されます。 WAVやその他の形式をご希望の場合は、依頼時にその旨をお伝えください。 追加料金なしで対応できる場合もございます。

決済は納品後の後払いとなります。安心してご利用ください。

いいえ、月額料金は一切かかりません！ 登録料・年会費も無料です。かかるのは依頼時の料金のみとなります。

X.comからログアウト

シークレットモード（PCに詳しい方向け）

💡 上級者向けのメリット：
• ログアウト不要で即座に切り替え
• ショートカットキーで素早く操作
• 通常モードとシークレットモードで2つのアカウントを同時使用

⚠️ 注意： この方法は「シークレットモード」の概念を理解している方向けです。初めての方は「基本的な方法」タブをご覧ください。

1
通常モードで1つ目のアカウントにログイン

いつも通りKimiLink Voiceにアクセスして、1つ目のアカウントでログイン

2
シークレットモードを開く

ショートカットキー：
• Chrome / Edge: Ctrl + Shift + N
• Firefox: Ctrl + Shift + P
• Safari: Cmd + Shift + N

3
シークレットモードで2つ目のアカウントにログイン

シークレットウィンドウで KimiLink Voice にアクセスし、2つ目のアカウントでログイン

4
完了！両方のアカウントを同時に使用

通常ウィンドウとシークレットウィンドウを切り替えるだけで、即座にアカウントを切り替えられます

ヒント

  • 最大のメリット： ログアウト不要！ウィンドウを切り替えるだけ
  • シークレットモードを閉じると、そのセッションは終了します
  • 3つ以上のアカウントを使う場合は、ブラウザプロファイル機能を利用してください
  • シークレットモードのショートカットキーを覚えておくと便利です

Chrome プロフィール機能

1
Chromeブラウザでプロフィールを追加

Chromeブラウザ右上のプロフィールアイコン（丸いアイコン）をクリック → 「追加」または「別のユーザーを追加」を選択

2
新しいChromeウィンドウが開きます

完全に独立したブラウザ環境として動作します

3
それぞれのプロフィールでログイン

各ウィンドウで異なるアカウントにログインできます

ヒント

  • 重要： KimiLink Voiceのプロフィールメニューではなく、Chromeブラウザ自体のプロフィール機能です（アドレスバーの右側、ブラウザの右上隅にあります）
  • プロフィールごとに名前とアイコンをカスタマイズできます
  • ブックマーク、履歴、パスワードも完全に分離されます
  • 複数のウィンドウを同時に開いて作業できます

Firefox コンテナタブ機能

1
Firefoxの拡張機能をインストール

Firefoxブラウザで「Multi-Account Containers」をFirefoxアドオンストアから無料でインストール

2
新しいコンテナタブを作成

タブの「+」ボタンを長押し → コンテナを選択

3
各コンテナで異なるアカウントにログイン

コンテナごとに独立したCookieを持ちます

ヒント

  • コンテナごとに色分けして管理できます
  • 同じウィンドウ内で複数アカウントを使い分け可能
  • 「仕事用」「プライベート用」など目的別に分けられます

Edge プロフィール機能

1
Edgeブラウザでプロフィールを追加

Edgeブラウザ右上のプロフィールアイコン → 「プロフィールの追加」を選択

2
新しいEdgeウィンドウが開きます

Chromeと同様に独立した環境で動作します

3
Microsoft アカウントと連携可能

設定を複数デバイス間で同期できます

ヒント

  • 重要： KimiLink Voiceのプロフィールメニューではなく、Edgeブラウザ自体のプロフィール機能です（アドレスバーの右側、ブラウザの右上隅にあります）
  • 職場アカウントと個人アカウントを分けて管理
  • EdgeとChromeは同じChromiumベース
  • Windows 11との統合が優れています

Safari プライベートウィンドウ

1
Safariブラウザでプライベートウィンドウを開く

Safariブラウザのメニュー「ファイル」 → 「新規プライベートウィンドウ」（⌘+Shift+N）

2
通常ウィンドウとプライベートウィンドウを両方使用

それぞれで異なるアカウントにログインできます

3
注意：プライベートウィンドウを閉じるとログアウト

セッションは保持されません

ヒント

  • MacユーザーにはChrome/Firefoxの方が便利
  • プライベートウィンドウは一時的な使用に適しています
  • iOS/iPadOSでも同様の方法が使えます
